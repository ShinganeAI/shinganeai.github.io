6. From Hardware to Software

With microprocessors (1970s), computing scaled down to the desktop. The focus shifted from building machines to writing algorithms.

This chart visualizes the conceptual shift in value generation. While hardware capacity (Moore's Law) grew exponentially, the complexity and value of software (Abstraction) became the dominant driver of innovation.
